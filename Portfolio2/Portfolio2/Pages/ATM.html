ATM

We were given the problem, to write a command-line cash machine(ATM) in C#. It is to have the following requirements:

  • The ATM is to be pre-stocked with the following denominations:
    • $100 - 10 Bills
    • $50 - 10 Bills
    • $20 - 10 Bills
    • $10 - 10 Bills
    • $5 - 10 Bills
    • $1 - 10 Bills
  • The application should take input from the standard input stream and have the following commands:
    • R - Restocks the cash machine to the original pre-stocked levels defined above.
    • W <dollar amount> - Withdraws that amount from the cash machine (e.g. W $145).
    • I <denominations> - Displays the number of bills in that denomination present in the ATM (e.g. I $20 $10 $50).
    • Q - Quits the application.
  • Other Instructions:
    • The withdrawals from the cash machine should dispense cash in the most efficient manner possible, with the least amount of bills. After a withdrawal, the program should display success or failure and the remaining balance in the cash machine. For an inquiry, the program should display the number of bills in the denominations specified. After a restock, the program should display the balance in the cash machine (same as after a withdrawal). If the input is not understood, “Invalid Command” should be displayed. No additional messages, prompts or errors should be displayed.
